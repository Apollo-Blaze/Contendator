<h2>🛠️ Installation Steps:</h2>

<p>1. installing flask for backend purposes</p>

```
pip install Flask
```

<p>2. for cross origin resource sharing</p>

```
pip install Flask-Cors
```

<p>3. installing openai module for communication</p>

```
pip install openai
```
